# Contributing Guide

## Scope

This repository contains infrastructure and documentation for FIN-CORE-01.

## Workflow

- Create changes on a branch.
- Keep changes small and well-documented.
- Update relevant docs and templates.

## Commits

- Use clear, imperative messages (e.g., "Add restore guide").

## Docs

- Follow `docs/STYLE_GUIDE.md`.
- Keep checklists concise and actionable.
