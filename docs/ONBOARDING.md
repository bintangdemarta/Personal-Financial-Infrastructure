# Onboarding Guide

## Goal

Get a new admin set up safely and consistently.

## Steps

1. Review `docs/ACCESS_CONTROL.md` and `docs/THREAT_MODEL.md`.
2. Ensure access via VPN or allowlisted IP.
3. Create account in Firefly III with least privilege.
4. Review backup/restore workflow and `docs/RESTORE_PLAYBOOK.md`.
5. Confirm admin can access logs and run health checks.
