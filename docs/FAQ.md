# FAQ

## What is this repository for?

It provides a documented, self-hosted personal finance stack using Firefly III.

## Where are backups stored?

Backups are stored in `./backups/` and should be copied offsite.

## How do I restore a backup?

Use `bash scripts/restore_db.sh backups/db_YYYYMMDD_HHMM.sql.gz`.

## What is the default branch?

`master`.
